Option 1 — Envoyer les nouveaux identifiants WiFi

L'ESP32 doit être connecté (statut vert) pour recevoir la commande.

Option 2 — Mode configuration (si ESP32 hors ligne)

  1. Débrancher et rebrancher l'ESP32
  2. Si le WiFi échoue, l'ESP32 crée un réseau "CTA-Config"
  3. Connecter votre téléphone/PC à CTA-Config
  4. Ouvrir http://192.168.4.1 dans le navigateur
  5. Saisir le nouveau SSID et mot de passe, puis valider
  6. L'ESP32 redémarre et se connecte au nouveau réseau
